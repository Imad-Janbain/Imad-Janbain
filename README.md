## 🧭 &nbsp; About

I build **end-to-end machine learning systems** — from data pipelines and model architecture to production deployment, monitoring, and reporting. My specialty is **time series forecasting** applied to hydrology and climate, combining modern deep learning with classical signal processing and LLM-powered explainability to deliver models that ship, not just publish.

Ph.D. in Deep Learning with published applied-research, currently postdoctoral researcher at **M2C – CNRS** — open to senior **ML Engineer**, **Applied Scientist**, and **Research Scientist** roles.

## 🚀 &nbsp; Flagship Projects

### 🌊 &nbsp; River Discharge Forecasting — End-to-End ML System

> **Production-grade forecasting platform** integrating classical, ML, and DL models with an LLM explainability layer.

**Engineering**
- **End-to-end pipeline** — ingestion → feature engineering → modeling → evaluation → reporting
- **Hybrid architecture** — ARIMA (baseline) · XGBoost (structured) · BiLSTM (sequential), with pluggable model layer
- **LLM reporting module** — Claude-powered natural-language summaries of forecasts for decision-makers
- **Dockerized & reproducible** — containerized deployment, clean module boundaries, ready for CI/CD

### 💧 &nbsp; Groundwater Forecasting — Wavelet + Deep Learning

> **Modular research-to-production pipeline** combining classical signal processing with recurrent deep learning.

**Engineering**
- **Modular design** — swappable deep architectures (GRU, LSTM, BiLSTM) behind a clean interface
- **Automated HPO** — Optuna-driven Bayesian search across architecture and training configs
- **Signal processing frontend** — MODWT decomposition (LA(8), multi-level) for non-stationary series
- **Full observability** — dedicated visualization module for reproducible result inspection
- **External data integration** — ERA5 climate reanalysis as exogenous drivers
